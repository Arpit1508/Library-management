# Library-management

A Java console-based Library Management System developed using Java, JDBC, and MySQL. This project allows users to manage library books with basic CRUD (Create, Read, Update, Delete) operations through a menu-driven interface.

🚀 Features
Add a new book
View all books
Search a book by ID
Update book details
Delete a book
Menu-driven console application
MySQL database integration using JDBC
🛠️ Technologies Used
Java
JDBC
MySQL
SQL
IntelliJ IDEA
